# 般若波羅蜜多心咒

```
gate gate paragate parasamgate bodhi svaha
```

### Chinese version

```
揭諦、揭諦，波羅揭諦，波羅僧揭諦，菩提薩婆訶。
```

### Japanese version

```
ぎゃ てい ぎゃ てい
は ら ぎゃ てい
は ら そう ぎゃ てい
ぼ じ そ わ か
```
