# 緣起咒

### Version.1

```
Om Yeh Dharma Hetu Prabhava
Hetun Tesam Tataghato Hyavadat
Tesam Cha Yo Nirodha
Evam Vadi Mahasharimahla Svaha
```

### Version.2
```
OM YE DHARMA HETU-TRABHAVA
HETUM TEKHAN TATHAGATO HYAVADAT
TESKHAN CAI YO NIRODHA
EVAM VADI MAHASRAMANAH YE SVAHA
```

### Chinese version

```
嗡耶 達爾瑪嘿德扎巴哇
嘿登得堪達塔嘎多
哈雅巴達 得堪匝友呢若達
誒旺巴德瑪哈夏爾瑪納 娑哈
```
